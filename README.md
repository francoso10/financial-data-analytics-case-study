# Financial Data Analytics Case Study

**ACC 430 Data Storytelling | Southern New Hampshire University**  
**Author:** Francisco Sousa

## Overview

This academic case study compares **Microsoft Excel Power Query / Power BI** and **Tableau** for recurring accounting analytics. The project evaluates how each platform supports data preparation, anomaly detection, duplicate-payment testing, job-cost and variance analysis, KPI reporting, and leadership communication.

## Business question

Which platform should an accounting department use as its primary tool for recurring analytics and controlled reporting?

## Recommendation

Use **Power Query / Power BI** as the primary accounting analytics platform because it better supports the end-to-end workflow from data preparation through repeatable reporting. Use **Tableau** as a supplemental tool when highly visual exploration or leadership-facing dashboards are the main goal.

## Analysis performed

- Data-quality profiling and field standardization
- Summary statistics and aggregations
- Outlier and cluster analysis
- Duplicate-payment exception testing
- Job-cost and variance analysis
- KPI thresholds and scorecard reporting
- Data storytelling for technical and non-technical audiences
